# [PRD] 국립중앙박물관 웹사이트 리디자인 : 예약 경험 최적화

**작성일:** 2026-04-10
**문서 상태:** 초안 (v1.0)
**프로젝트 성격:** 공공기관 웹사이트 사용성 및 접근성 개선

---

## 1. 프로젝트 배경
국립중앙박물관은 전 연령층이 이용하는 국가적 문화 서비스이나, 현재 웹사이트는 복잡한 정보 구조와 시각적 가이드의 부재로 인해 핵심 기능인 '관람 예약'에 대한 접근성이 매우 낮음. 특히 고령층 및 IT 취약계층의 디지털 소외 현상을 해결하고, 박물관의 브랜드 가치를 올바르게 전달하기 위한 리디자인을 추진함.

## 2. 문제 정의 (Problem Definition)
* **발견의 어려움:** 예약 버튼이 텍스트 링크 형태로 하단에 배치되어 있어, 첫 화면에서 과업을 시작하기 어려움.
* **인지적 혼선:** 동일한 정보가 배너, 캐러셀 등에 반복 노출되어 사용자에게 불필요한 인지 부하를 제공함.
* **낮은 조작 편의성:** 클릭 타겟 사이즈가 작고 호버(Hover) 효과에 의존하고 있어, 마우스 조작이 미숙한 사용자의 오클릭 유발.
* **비효율적 여정:** 예약 버튼 클릭 후에도 다시 원하는 전시를 찾아야 하는 등 단계별 이탈 요소가 많음.

## 3. 핵심 UX 가설
> **"만약 우리가 단순 텍스트 링크 형태의 예약 버튼을 명확한 버튼 타입으로 변경하고 첫 화면(Above the fold)에 배치한다면, 사용자들은 즉각적으로 예약 페이지로 진입하는 긍정적인 결과를 얻을 것이다. 왜냐하면 버튼의 시각적 기표(Signifier)가 강화되어 '누를 수 있는 요소'임을 본능적으로 인지하기 때문이다."**

## 4. 타겟 페르소나
* **이름:** 박영희 (68세, 전업주부)
* **IT 숙련도:** 최하 (카카오톡, 유튜브 시청 위주)
* **목표:** 손주를 위해 스스로 '특별 어린이 체험 전시'를 예약하고 싶어 함.
* **핵심 Pain Point:** 스크롤 조작 미숙, 작은 텍스트 인식 불가, 손떨림으로 인한 클릭 실패.

## 5. 핵심 요구사항 및 기능 명세

### 5.1 인터페이스 가시성 강화 (Visibility)
* **[REQ-01] 메인 CTA 버튼화:** '예약하기'를 텍스트가 아닌 컬러 배경과 명확한 대비를 가진 **버튼 타입(Solid Button)**으로 변경.
* **[REQ-02] 최상단 고정 배치:** 사이트 접속 시 스크롤 없이 바로 인지할 수 있도록 **Hero Section** 내에 배치.
* **[REQ-03] 타겟 사이즈 최적화:** 모든 인터랙션 요소의 크기를 **최소 44x44px 이상**으로 확보(WCAG 2.5.5 준수).

### 5.2 사용자 여정 효율화 (User Journey)
* **[REQ-04] 원스톱 예약 시스템:** 예약 버튼 클릭 시 페이지 이동 없이 **레이어 팝업**을 통해 전시-날짜-회차를 한 번에 선택 가능하도록 설계.
* **[REQ-05] 인지적 피드백 제공:** 클릭 시 시각적 변화를 명확히 하고, 다음 단계로 이동하는 아이콘(화살표 등)을 추가하여 심리적 안정감 제공.

### 5.3 시각 디자인 및 접근성 (Accessibility)
* **[REQ-06] 명도 대비 개선:** 배경과 텍스트의 명도 대비를 **4.5:1 이상**으로 유지하여 가독성 확보.
* **[REQ-07] 가독성 중심 타이포그래피:** 고령층을 고려하여 본문 폰트 크기를 상향 조정하고 가독성이 검증된 서체 적용.

## 6. TO-BE Task Flow (성공 경로)
1. **진입:** 메인 접속 후 1초 이내에 상단 '예약하기' 버튼 발견.
2. **액션:** 큰 버튼 영역을 클릭하여 예약 팝업 호출.
3. **선택:** 팝업 내 큼직한 전시 카드를 보고 '어린이 박물관' 선택.
4. **설정:** 시인성이 높은 달력 UI에서 날짜와 시간 선택.
5. **완료:** '예약 완료' 버튼 클릭 후 명확한 피드백(알림창) 확인.

## 7. 성공 측정 지표 (KPI)
* **예약 전환율 (CVR):** 메인 페이지 방문 대비 예약 완료 성공 비율 증가.
* **과업 완수 시간 (ToT):** 접속 후 예약 완료까지 걸리는 시간 단축 (현재 대비 50% 이상 단축 목표).
* **고령층 사용성 점수:** 타겟 페르소나 대상 UT 진행 시 '혼자서 수행 가능' 답변 비중 80% 이상 확보.
